# mysql2json

dump sql result from mysql as json.
php version of [mysql2xxxx](https://github.com/seamusabshere/mysql2xxxx)

## Installation

require: php

## Usage

php mysql2json.php -hlocalhost -Dtest -uroot -proot --execute="select * from feeds"
